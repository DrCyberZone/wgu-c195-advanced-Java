How to use the program:

The program opens the log-in screen. To access the program, use "admin" or "test" as the username/password.
After logging in, a Pop-Up appears to show if there is an appointment within 15 minutes of the user's current time at the main menu screen.
Users can add, update, and delete customers and appointments from the main menu.
Users can view all appointments or sort by current week/month.
Users can generate reports by clicking on the "Reports" button.



Description:

This app was designed for a company to schedule appointments for their clients.
Customers can schedule appointments and view and edit them using the GUI application. This app will remind the user of a position within the next 15 minutes.
This app can also change the software's language depending on the user's system language.
